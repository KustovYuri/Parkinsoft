Для запуска бекенда необходимо установить и настроить PostgreSQL (https://www.postgresql.org/download/windows/). При установке запомните пароль, порт оставляем по умолчанию. В PgAdmin создаем базу данных с названием myDb
После создания базы данных заходим в бекенде в src/main/resources/application.yml и меняем ${DB_URL} на jdbc:postgresql://localhost:5432/myDb и ${DB_PASSWORD} на пароль из базы данных
Для начальной инициализации базы данных можно использовать файл, находящийся тут Parkinsoft/backend/src/main/kotlin/com/parkinsoft/backend/database_utils/DataInitializer.kt и раскоментировать строчки 25, 26, 27
Если потребуется ввести новые данные врача то можно добавить его в doctorInit на 632 строчке
Важно: после запуска бекенда необходимо снова закоментировать строчки 25, 26, 27 иначе данные будут дублироваться при каждом перезапуске
Так же для сброса базы данных можно в src/main/resources/application.yml заменить ddl-auto: update на ddl-auto: create, тогда при каждом перезапуске база данных будет сбрасываться

Для запуска мобильного приложения необходимо заменить ip адрес на локальный ip адрес устройства, на котором запускается приложение, посмотреть его можно в консоли, написав ipconfig (IPv4-адрес)
Полученный адрес необходимо вставить в app/src/main/java/com/farma/parkinsoftapp/di/NetworkModule.kt, строка 77
